Blcok & Inline 태그
    - Block 태그 : 문단을 표현하는 태그
        
        예) p, div, form … 태그
        
        - 태그와 태그 사이에 줄개행
        - 너비, 높이를 스타일로 지정 가능, 내외부 여백 모두 적용가능

    - Inline 태그 : 문장을 표현하는 태그
        
        예) a, span, i, … 태그
        
        - 태그와 태그 사이에 줄개행 하지 않음
        - 너비, 높이를 스타일로 지정 불가, 외부 여백은 좌우만 적용 가능

이미지를 설정하는 태그
<img>

-src : 이미지 경로
-width : 이미지의 너비
-height : 이미지의 높이
-너비, 높이를 함께 설정하면 이미지의 왜곡이 발생할 가능성이 높다.
-너비, 높이 중에서 한가지 기준으로 설정하는게 좋다!
(한가지만 설정하면 나머지는 원본 비율에 맞춰 자동으로 설정됨)
-alt : 대체 텍스트 (이미지가 출력될 수 없는 경우 대신 출력되는 문자) / (이미지에 대한 설명)

참고1) 모니터 단위 화소 - 점 1개가 1px(픽셀)
참고2) 이미지 경로 즉 경로에는 절대 경로, 상대 경로가 존재
	절대 경로 - 전체 경로
	상대 경로 - 기준 경로의 상대적인 경로 지정
		기준경로 : 웹 주소 기준, 파일 명 제외 경로

속성

action : 양식을 처리할 제출 경로(서버쪽 URL)

method : 처리 방식 - GET(기본값), POST, PUT, PATCH, DELETE, OPTIONS, HEAD

input 태그

type - text - 한줄 텍스트(기본값)

        - password : 비밀번호 입력 텍스트 (감춤 처리)

        - radio : 여러개 중에서 한개를 선택, name 속성으로 선택 그룹을 묶는다.

        - label 태그로 감싸면 텍스트를 클릭해도 선택이 된다.

        - checked : 선택된 상태로 변경

        (input까지 1줄씩 감싸야함!)

        - checkbox : 여러개 중에서 여러개를 선택한다.

              - checked : 선택된 상태로 변경

        - submit : 양식 제출 버튼

               - button :  태그로도 교체 가능

        - reset, button

        - number : 숫자만 입력 가능

        - email : 이메일 형식의 문자만 입력 가능, 형식이 맞지 않으면 양식 제출 불가

        - date : 날짜 형식으로 입력 받는 경우, 선택할 수 있는 달력이 노출

        - time : 시간 형식으로 입력 받는 경우

        -datetime : 날짜와 시간

        -range : 범위 선택

        -max : 최대값, min : 최소값, step : 증감 단위

        - color : 색상 선택 

        - hidden : 사용자가 입력하는 입력 항목이 아니다! 양식에 고정해서 서버에 전달하는 값

        - file : 파일 선택, 클릭시 파일 탐색기가 노출

        파일의 형식을 제한하는 경우

        - accept=”image/*” : 이미지 형식의 파일만 선택 가능 제한

        - multiple : 파일 여러개 선택 가능

value : 값을 결정, 사용자가 입력을 하는 태그인 경우는 입력값이 자동으로 value로 설정

name : 전송할 데이터의 항목명, 없으면 데이터의 전송과 수시니 불가!

required : 양식을 제출할때 필수로 요구되는 항목

readonly : 읽기 전용, 사용자가 직접 입력 X, 데이터가 전송가능

disabled : 입력 불가 처리, 데이터 전송 불가

placeholder : 입력 값이 없는 경우 대체 안내 문구

minlength : 최소 길이 한정

maxlength : 최대 길이 한

button

type - submit : 기본 값, 양식 제출

        - reset : 입력 항목 초기화

        - button : 일반적인 버튼

textarea : 여러줄 텍스트를 입력받는 경우

 - rows : 행의 갯수

 - cols : 열의 갯수

select : 여러개 항목중에서 한개 또는 그 이상을 선택할때

속성 :

 - multiple : 여러개 선택 가능

 - size : 한번에 보이는 항목 갯수

 - option : 선택 항목

 - selected : 선택 처리 (~checked)

**CSS**

- CSS(Cascading Style Sheet)는 HTML과 같은 마크업 언어로 작성된 문서에 색상, 폰트, 각 요소의 위치 변경, 백그라운드 및 간단한 애니메이션(transition) 등의 효과를 주어 보다 보기 좋게 꾸며 주는 역할을 합니다.

1. HTML에서 CSS를 적용하는 방법

1. 속성으로 적용
    
    style 속성
    
    <p style=”font-size: 11px”>…</p>
    
2. style 태그안에 적용
    
    <style>
    
    선택자 {
    
    속성: 값;
    
    속성: 값;
    
    …
    
    }
    
    </style>
    

참고)

; - 문장의 종료

1. 외부 파일로 적용
    
    CSS 파일은 확장자 .css
    
    <link rel=”stylesheet” type=”text/css” href=”스타일 파일 경로”>
